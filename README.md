# Idea Picker 🎲

A simple web app to help you pick a random project idea and avoid decision paralysis.

## Features
- Add unlimited ideas
- Random picker
- Remove used ideas
- Saves data with localStorage

## Tech
- HTML, CSS, JavaScript
